# Requirement
 
* webpack 5.72.1
* webpack-cli 4.9.2
* webpack-dev-server 4.9.1
* typescript 4.7.2
* ts-loader 9.3.0
* .
* .
Please see "package.json".

# Installation
```bash
# install dependencies
npm install
```
 
# Usage
 
```bash
# app build
npm run build

# serve with hot reload
npm run start
```
